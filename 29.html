Chapter 12, Iterators and Generators Explains how the for/of loop works and how you can make your own classes iterable with for/of. It also covers generator functions and the yield statement. Chapter 13, Asynchronous JavaScript This chapter is an in-depth exploration of asynchronous programming in Java‐ Script, covering callbacks and events, Promise-based APIs, and the async and await keywords. Although the core JavaScript language is not asynchronous, asynchronous APIs are the default in both web browsers and Node, and this chapter explains the techniques for working with those APIs. Chapter 14, Metaprogramming Introduces a number of advanced features of JavaScript that may be of interest to programmers writing libraries of code for other JavaScript programmers to use. Chapter 15, JavaScript in Web Browsers Introduces the web browser host environment, explains how web browsers exe‐ cute JavaScript code, and covers the most important of the many APIs defined by web browsers. This is by far the longest chapter in the book. Chapter 16, Server-Side JavaScript with Node Introduces the Node host environment, covering the fundamental programming model and the data structures and APIs that are most important to understand.
