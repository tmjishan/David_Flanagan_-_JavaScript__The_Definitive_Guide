Like regular functions, however, we can also define generators in expression form. Once again, we just put an asterisk after the function keyword: const seq = function*(from,to) { for(let i = from; i <= to; i++) yield i; }; [...seq(3,5)] // => [3, 4, 5] In classes and object literals, we can use shorthand notation to omit the function keyword entirely when we define methods. To define a generator in this context, we simply use an asterisk before the method name where the function keyword would have been, had we used it: let o = { x: 1, y: 2, z: 3, // A generator that yields each of the keys of this object *g() { for(let key of Object.keys(this)) { yield key; } } }; [...o.g()] // => ["x", "y", "z", "g"] Note that there is no way to write a generator function using arrow function syntax.
