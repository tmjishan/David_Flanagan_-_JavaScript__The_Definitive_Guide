parentNode This property of an element refers to the parent of the element, which will be another Element or a Document object. children This NodeList contains the Element children of an element, but excludes non- Element children like Text nodes (and Comment nodes). childElementCount The number of Element children. Returns the same value as children.length. firstElementChild, lastElementChild These properties refer to the first and last Element children of an Element. They are null if the Element has no Element children. nextElementSibling, previousElementSibling These properties refer to the sibling Elements immediately before or immediately after an Element, or null if there is no such sibling. Using these Element properties, the second child Element of the first child Element of the Document can be referred to with either of these expressions: document.children[0].children[1] document.firstElementChild.firstElementChild.nextElementSibling (In a standard HTML document, both of those expressions refer to the <body> tag of the document.)
