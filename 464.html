In HTML, any attribute whose name is lowercase and begins with the prefix ‚Äúdata-‚Äù is considered valid, and you can use them for any purpose. In the DOM, Element objects have a dataset property that refers to an object that has properties that correspond to the data- attributes with their prefix removed. Thus, dataset.x would hold the value of the data-x attribute. Hyphenated attributes map to camelCase property names: the attribute data-section-number becomes the property dataset.sectionNumber.
