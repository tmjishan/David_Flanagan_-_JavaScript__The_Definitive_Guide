This is easy to do with the classList property of Element objects, as explained in “The class attribute” on page 445. In general, we can’t create a different stylesheet class for each possible position of the tooltip, so the classList property won’t help us with positioning. In this case, we need to script the style attribute of the tooltip element to set inline styles that are specific to that one element. The DOM defines a style property on all Element objects that correspond to the style attribute. Unlike most such properties, however, the style property is not a string. Instead, it is a CSSStyleDeclaration object: a parsed representation of the CSS styles that appear in textual form in the style attribute.
