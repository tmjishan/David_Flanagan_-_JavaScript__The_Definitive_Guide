If a custom element class defines a static observedAttributes property whose value is an array of attribute names, and if any of the named attributes are set (or changed) on an instance of the custom element, the browser will invoke the attributeChanged Callback() method, passing the attribute name, its old value, and its new value. This callback can take whatever steps are necessary to update the component based on its attribute values. Custom element classes can also define whatever other properties and methods they want to. Commonly, they will define getter and setter methods that make the ele‐ ment’s attributes available as JavaScript properties.
